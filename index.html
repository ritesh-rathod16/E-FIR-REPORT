File Police Complaints Online

Report incidents, file FIRs, and track your complaints through our secure online portal. Accessible anytime, anywhere.

Services We Provide

Cognizable Offences FIR

File an FIR for serious crimes where police can arrest without a warrant (e.g., murder, rape, theft, robbery).

Non-Cognizable Report (NCR)

Report less serious crimes where police need a magistrate's order to investigate (e.g., simple assault, defamation).

Zero FIR

File an FIR at any police station regardless of jurisdiction. It will be transferred to the appropriate station.

Cyber Crime

Report cyber crimes including online fraud, hacking, identity theft, and cyber harassment.

Women's Safety

Report crimes against women including harassment, domestic violence, and abuse.

Counter FIR

File a counter complaint in situations where two parties accuse each other of offences from the same incident.

FIR Classification Overview

Based on Cognizability

Cognizable Offences: Serious crimes where police can arrest without warrant and investigate without magistrate's permission (e.g., murder, rape, theft).

Non-Cognizable Offences: Less serious crimes where police need magistrate's order to investigate (e.g., simple assault, defamation).

Special Categories

Zero FIR: Can be filed at any police station regardless of jurisdiction.

E-FIR: Filed online through official police portals.

Counter FIR: Filed by opposing parties in the same incident.
